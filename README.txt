Скрипт прокси
Версия 1.2

Автор: Васянович Антон (antonvasyanovich1@gmail.com)

Описание
	Данный скрипт реализует прокси-сервер на компьютере, с которого скрипт запускался

Требования
	Python версии не ниже 3.6

Состав
	Консольная версия: proxymain.py
			   server.py
			   unittests.py

Запуск
	Запустить скрипт через терминал с использованием ключа -p или --port. При необходимости воспользоваться справкой по ключу -h либо --help.
